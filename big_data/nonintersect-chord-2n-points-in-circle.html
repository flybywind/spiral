圆内2n个点求n种不交叉弦的画法

ref: http://en.wikipedia.org/wiki/Motzkin_numbe

思路:

这个又是迭代思路的经典案例。(之前面试还遇到一个爬楼梯的问题，结果就是Fabonaqi数列）
首先，选择2n个里面其中一个点，然后，画一条弦S，使得这个弦将点集分成2个集合，每个集合里面也是偶数个点，设为2p，2q
那么有 
				2p + 2q + 2 = 2n

这样问题就分解为，有多少种画S的方式，答案是n种， 所以 
				F(2n) = F(2n - 2)*F(0) + F(2n - 4)*F(2) + ... F(2)*F(2n - 4) + F(0)*F(2n - 2)
注意 F(2n - 2)*F(0)和 F(0)*F(2n - 2)是2种不同的选择方式 所以F(2n)可以进一步简化为： 
				F(2n) =
					  2*{ F(2n - 2)*F(0) + F(2n - 4)*F(2) + ... F(n)*F(n-2) }, if n is even
					  2*{ F(2n - 2)*F(0) + F(2n - 4)*F(2) + ... F(n + 1)*F(n - 3) } + F(n - 1)F(n - 1), if n is odd

点评：

这个问题的技巧在于，需要构造一个和父问题类似的子问题。然后再用迭代
